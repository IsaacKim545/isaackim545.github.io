How It Works

사용법은 정말 간단합니다

별도의 설정이나 학습 없이, 열자마자 바로 쓸 수 있어요.

1

링크를 열기

PC, 스마트폰, 태블릿 어디서든 브라우저로 접속하면 바로 가계부가 열립니다. 앱 설치나 회원가입은 필요 없습니다.

2

수입·지출 입력하기

"추가" 탭에서 금액을 입력하고, 카테고리를 선택하고, 필요하면 메모를 남기세요. 수입과 지출을 토글로 간편하게 전환할 수 있습니다.

3

한눈에 확인하기

"홈" 탭에서 이번 달 잔액과 최근 거래를 보고, "통계" 탭에서 카테고리별 지출 비율과 6개월 추이를 차트로 확인하세요.

4

앱처럼 설치하기 (선택)

폰 홈 화면에 추가하면 네이티브 앱처럼 전체화면으로 열리고, 인터넷이 없는 오프라인 환경에서도 사용할 수 있습니다.

Privacy

개인정보, 어디에도 올라가지 않습니다

🔒 100% 로컬 저장

이 가계부 앱은 별도의 서버를 사용하지 않습니다. 입력하는 모든 데이터는 사용자의 기기 브라우저(IndexedDB)에만 저장됩니다. 외부 서버로 전송되거나 수집되는 정보는 전혀 없습니다.

회원가입이 없기 때문에 이메일, 전화번호, 이름 등의 개인정보를 입력할 필요도 없습니다. 누가 쓰는지 앱이 알 수 없는 구조입니다.

다만, 브라우저 데이터를 삭제하면 가계부 데이터도 함께 삭제되니 이 점만 주의하세요.

Install

폰에서 앱처럼 사용하기

홈 화면에 추가하면 앱 아이콘이 생기고, 전체화면으로 실행됩니다. 오프라인에서도 사용할 수 있어요.

Android (Chrome)
  1. Chrome 브라우저로 가계부 링크 열기
  2. 오른쪽 상단 메뉴(⋮) 탭
  3. "앱 설치" 또는 "홈 화면에 추가" 선택
iPhone (Safari)
  1. Safari 브라우저로 가계부 링크 열기
  2. 하단 공유 버튼(□↑) 탭
  3. "홈 화면에 추가" 선택
FAQ

자주 묻는 질문

정말 무료인가요?

네, 완전 무료입니다. 유료 결제나 인앱 구매 같은 건 없습니다.

데이터가 사라지지는 않나요?

일반적인 사용에서는 사라지지 않습니다. 다만 브라우저의 전체 데이터를 삭제하거나 시크릿 모드에서는 데이터가 유지되지 않으니 주의하세요.

여러 기기에서 동기화되나요?

현재는 각 기기별로 독립적으로 저장됩니다. 폰에서 입력한 내용이 PC에 자동으로 반영되지는 않습니다.

인터넷 없이도 사용할 수 있나요?

네. 한 번 접속한 뒤에는 오프라인에서도 사용할 수 있습니다. 홈 화면에 앱으로 설치하면 더 안정적으로 동작합니다.

어떤 브라우저에서 되나요?

Chrome, Safari, Edge, Firefox 등 대부분의 최신 브라우저에서 사용할 수 있습니다. Chrome과 Safari를 권장합니다.
